## 🌐 Live Demo

https://lab-14d8c.web.app/

---

## Overview

This project was developed as part of a group assignment at **École Multimédia**, in collaboration with a creative team.

I was responsible for the **web development of the project**.

The website is inspired by the **visual and artistic universe of Isaiah Rashad**, focusing on music-driven storytelling, immersive atmosphere, and emotional visual direction.

---

## Concept

The project explores an immersive web experience built around an artistic universe inspired by Isaiah Rashad’s music.

It combines:
- Visual storytelling
- Modern UI/UX design
- Animations
- Interactive navigation
- Atmospheric and emotional direction

---

## Tech Stack

- React (Vite)
- Firebase (backend / database / hosting)
- Framer Motion (animations)
- React Scroll (smooth navigation)
- Modern responsive CSS

---

## Features

- Fully responsive and optimized website
- Smooth animations and transitions
- Interactive scroll-based navigation
- Firebase integration
- Dynamic content management system
- Event/date management through an admin interface

---

## Admin Panel

An admin interface was built to manage website content in real time, including:

- Add / edit / delete event dates
- Update displayed information dynamically
- Firebase real-time synchronization
